# 克隆项目
外网用户：

```
git clone https://github.com/ImGili/GU.git
```

内网用户：

```
git clone https://gitee.com/ImGili/GU.git
```

# 构建项目
```
mkdir build
cd build
cmake ..
cmake --build ./ --config Release --target GU -j 10
```
